# Rust Interpreter

## Features

Currently, this interpreter lexes the following tokens

- '='
- '+'
- '-'
- '!'
- '\*'
- '/'
- '<'
- '>'
- ','
- ';'
- '('
- ')'
- '{'
- '}'
- 'let'
- 'fn'
- 'if'
- 'else'
- 'return'
- 'true'
- 'false'
- Identifiers containing alphabetic characters
- non negative 32 bit integers
